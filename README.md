## How to use

```shell
Usage:
  rgif [flags] [query]

Flags:
  -u    print url only
  -o    open in default browser

Examples:
  rgif oh boy
  rgif -o whatever
  rgif -u "can't touch this!"
```

## Installation

```shell
go get -u github.com/michaelhue/rgif
```
